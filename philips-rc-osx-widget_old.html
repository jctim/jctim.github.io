Remote control dashboard widget for Philips TV


Developed with help of:

  • Dashcode IDE for OS X (current version 3.0.5 is not compatible with Mavericks yet)
  • JointSPACE API
  • curl to make post requests
  • Icons set by Icons8

Known issues:

  • Buttons "3D" and "List" do not work because they are not mapped in JointSPACE API. But they do in Philips MyRemote ios app, so the fix is just a matter of time
