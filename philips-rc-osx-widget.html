Remote control OSX dashboard widget for Philips TV

If you are tired after very long workday and decided to spend some time with your Phlilips TV with your macbook on your laps, this small widget will help you to make interactions with your TV-set without help of real RC :)

Sources Widget View on GitHub

Tools and libs

  • Dashcode IDE for OS X (current version 3.0.5 is not compatible with Mavericks yet)
  • JointSPACE API
  • curl to make post requests
  • Icons set by Icons8

Known issues

  • Buttons "3D" and "List" do not work because they are not mapped in JointSPACE API. But they do in Philips MyRemote ios app, so the fix is just a matter of time
